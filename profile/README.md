 Olá, somos estudantes finalistas tentando mudar o mundo com a tecnologia! 👋
 
Sobre
Somos uma organização  estudantes finalistas do ensino médio focadas no desenvolvimento de soluções tecnológicas, com o objetivo de criar sistemas digitais úteis e eficientes.
Este espaço reúne os nossos projetos, incluindo o desenvolvimento de um Prontuário Médico Digital, como projecto final de curso.

 Projeto Principal: Prontuário Médico Digital
 
Sistema desenvolvido para gerir informações médicas de forma digital, permitindo:

-Registro de pacientes
- Histórico clínico
- Acesso seguro aos dados

DESCRIÇÃO DO PROJECTO
Esta organização consiste em desenvolver um sistema de prontuário médico virtual, desenvolvido para armazenar, organizar e facilitar o acesso a informações médicas e pacientes
Esse sistema permite que profissionais de saúde registrem e consultem dados clínicos de forma digital, substituíndo processo manuais
